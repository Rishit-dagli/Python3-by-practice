### Problem Statement:
The prime factors of 13195 are 5,7,13 and 29. What is the largest prime factor
of a given number N?

e.g. for 10, largest prime factor = 5. For 17, largest prime factor = 17.

Sample Test Case:
 
 Input - 13195<br>
 Output - 29

 Input - 10<br>
 Output - 5
  
 Input - 17<br>
 Output - 17
 
 Input - 3.4<br>
 Output - 3
